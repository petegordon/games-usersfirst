Frequently Asked Questions

Open tandemonium.jimandi.love in your mobile or desktop browser. On mobile, tilt your phone to steer and tap the pedal buttons to ride. On desktop, use A/D keys or a gamepad. No download needed!
Choose "Ride Together" from the lobby. One player creates a room and shares the room code. The other player enters the code to join. The game connects you directly via peer-to-peer — no account required.
Tandemonium works in any modern browser (Chrome, Safari, Firefox, Edge) on phones, tablets, and desktops. For the best mobile experience, we recommend a recent iPhone or Android device with gyroscope support.
Make sure you've granted motion/orientation permissions when prompted. On iOS, you may need to tap "Allow" on the permission dialog. If it still doesn't work, try the sensitivity slider in the game or switch to keyboard/gamepad controls.
Peer-to-peer connections can sometimes be blocked by strict firewalls or network configurations. Try switching to a different Wi-Fi network or using mobile data. Make sure both players are using the same room code and that one player has selected "Captain" and the other "Stoker."
Safety Mode limits how far the bike can lean, making it harder to crash. It's enabled by default for a more forgiving experience. You can toggle it off in the game controls if you want the full physics challenge.
Yes! Tandemonium is free to play in your browser. A Steam version with additional features is coming soon.
Some bikes are unlocked by completing specific challenges — for example, winning races with certain bike combinations. Check your achievements screen for hints on what to unlock next.
